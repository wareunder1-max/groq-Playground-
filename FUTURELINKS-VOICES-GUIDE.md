## How to Change Voice

1. **In the web interface**:
   - Select "FutureLinks.ai" as TTS provider
   - Find the "Voice:" dropdown
   - Select your preferred voice
   - Voice will be saved automatically

2. **Voice will apply to**:
   - Test button
   - Conversation mode
   - All TTS responses

## Tips for Best Results

### 1. Match Voice to Content
- Use female voices for conversational AI
- Use male news voice for formal announcements
- Use storytelling voice for narratives

### 2. Adjust AI Persona
When changing voice gender, update the AI system prompt to match:
- Female voice → Use feminine pronouns in Urdu (میں ہوں، مجھے، میری)
- Male voice → Use masculine pronouns in Urdu (میں ہوں، مجھے، میرا)

### 3. Test Different Voices
- Try different voices for different use cases
- Some voices work better for certain content types
- User preference varies

### 4. Language Matching
- Use Urdu voices for Urdu content
- Use Sindhi voices for Sindhi content
- Use Balochi voices for Balochi content
